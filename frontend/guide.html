Lesson Booking 사용자 설명서

학생/선생님 모두가 예약 상태를 빠르게 파악하고 조작할 수 있도록 설계된 화면 사용 안내입니다.

캘린더 기반 예약 역할별 대시보드 취소 마감 정책 반영 실시간 API 로그
예약 화면으로 이동 API Health 확인

1) 어떤 기능을 쓸 수 있나요?

현재 MVP에서 실제 사용 가능한 기능 목록입니다.

학생 기능

  • 선생님 선택 후 월간 캘린더에서 가능한 날짜 탐색
  • 일별 시간 슬롯 즉시 예약
  • 내 예약 목록 조회 및 상태 확인
  • 정책 허용 범위 내에서 예약 취소

선생님 기능

  • 주간 반복 시간표(Availability) 등록/수정/삭제
  • 특정 날짜 예외(종일/부분 차단) 등록/삭제
  • 본인 예약 목록 조회
  • 필요 시 예약 취소 처리

2) 학생 사용 흐름

처음 사용하는 학생 기준 추천 순서

Step 1. 로그인

학생 계정(`student@example.com`)으로 로그인하면 학생 대시보드가 열립니다.

Step 2. 선생님/월 선택

상단에서 선생님을 선택하고 `이전 달`, `다음 달`, `Today`로 월을 이동합니다.

Step 3. 슬롯 예약

일별 슬롯 목록에서 `예약` 버튼을 눌러 즉시 예약을 생성합니다.

Step 4. 예약 관리

`내 예약 목록`에서 BOOKED 상태를 확인하고 필요 시 취소합니다.

3) 선생님 사용 흐름

운영 화면에서 일정 공급(시간표)을 먼저 만들어야 학생에게 슬롯이 보입니다.

Step 1. 시간표 등록

요일/시작/종료/활성 여부를 입력해 반복 가능한 레슨 시간을 등록합니다.

Step 2. 예외 등록

휴무일은 날짜만 입력(종일 차단), 부분 차단은 시작/종료 시간을 함께 입력합니다.

Step 3. 예약 모니터링

`내 예약 목록`에서 학생 예약을 확인하고 필요 시 취소 처리합니다.

4) 화면 요소 설명

사용자가 자주 확인하는 핵심 UI 의미

요소 의미
월간 캘린더 날짜 하단 숫자 해당 날짜의 예약 가능한 슬롯 수
Today 버튼 현재 달/오늘 날짜 기준으로 바로 이동
예약 가능만 보기 이미 예약된 슬롯을 숨기고 가능한 시간만 필터링
상단 요약 카드 이번 달 슬롯 수, 가능한 슬롯 수, 활성 예약 수를 즉시 파악
API 로그 요청/응답 코드를 바로 확인해 문제 상황을 추적

5) 정책 안내

서버 정책에 따라 동작이 제한되는 구간

  • 예약은 선생님별 예약 가능 기간(기본 30일) 내에서만 생성됩니다.
  • 학생 취소는 수업 시작 전 컷오프 시간(기본 6시간)을 지나면 거절됩니다.
  • 동일 슬롯에 대해 동시에 예약하면 한 건만 성공하고 나머지는 충돌(409)로 처리됩니다.
  • 과거 시간 예약은 허용되지 않습니다.
운영 팁: 먼저 선생님 화면에서 시간표/예외를 정리한 뒤 학생 화면으로 검증하면 슬롯 노출 문제를 빠르게 확인할 수 있습니다.
