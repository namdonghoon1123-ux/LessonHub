## 이슈 / 리스크
- `docker-compose up`만으로는 마이그레이션/시드가 자동 적용되지 않는다. fresh volume 상태에서는 `npm run migrate`, `npm run seed` 전까지 로그인/주요 API가 실패한다.
- `docker-compose exec -T backend npm run seed`가 `npm run migrate` 직후 첫 시도에서 간헐적으로 `relation "users" does not exist`를 낼 때가 있었다. 같은 스택에서 재시도하면 성공했고, 이후 DB 상태도 정상으로 확인됐다. 컨테이너 exec/초기화 타이밍인지 추가 확인이 필요하다.
- Playwright MCP는 이 환경에서 `/.playwright-mcp` 경로 생성 실패(읽기 전용 파일시스템)로 브라우저 자동화가 막혀 있다. 이번 턴의 UI 스모크는 HTTP/API 기준 검증에 머물렀다.
