## 이슈 / 리스크
- Homebrew 기반 Docker Desktop 자동 설치는 로컬 권한(`sudo` 비밀번호) 이슈로 자동화 불가
- macOS 메타파일(`._*`)이 누적되면 Docker Desktop BuildKit에서 `xattr ... operation not permitted` 오류가 재발할 수 있음
